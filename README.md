#  Emotion Based Music Player

This project detects facial emotions using a webcam and plays YouTube music based on detected emotion.

# Features
- Real-time face detection
- Emotion recognition using deep learning model
- Auto music recommendation based on emotion
- Auto-stop timer + manual stop

# Tech Stack
- Python
- OpenCV
- TensorFlow / Keras
- Webbrowser module

# How to Run

```bash
pip install -r requirements.txt
python main.py
